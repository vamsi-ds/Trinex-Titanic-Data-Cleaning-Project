# Titanic Data Cleaning & Visualization Project

## Project Overview

This project performs data cleaning, preprocessing, and visualization on the Titanic Dataset using Python.

The objective is to clean raw data, handle missing values, identify duplicates and outliers, and generate meaningful visual insights.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset

Titanic Dataset

## Tasks Performed

* Dataset Loading
* Data Inspection
* Missing Value Handling
* Duplicate Removal
* Outlier Detection
* Data Visualization
* Correlation Analysis

## Visualizations

* Survival Count Plot
* Gender vs Survival Plot
* Boxplots
* Correlation Heatmap

## Key Insights

* Female passengers had higher survival rates.
* Missing values were successfully handled.
* Data quality improved after preprocessing.
* Important trends and relationships were identified.

## Conclusion

This project demonstrates practical data cleaning and visualization skills using Python and Data Science libraries.
